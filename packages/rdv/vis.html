How to trust this

Check it, or rebuild it

Verify the shipped code against its contract — deterministic, no tokens. Fails loudly if the bytes don't match the held-out oracle or the recorded hashes.
rdv check .
Rebuild it yourself from the spec — torch some tokens, regenerate src/ from sir/ + oracles/ via N isolated quorum workers. Trust your own build, not the publisher's.
rdv resynth . # prepare worker prompts rdv resynth . --apply # grade, pick quorum, update src/ + manifest
